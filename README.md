# 🎮 W3Labs Store

A **W3Labs Store** é uma plataforma de catálogo e gerenciamento de arquivos (jogos, ISOs, softwares) desenvolvida para uso interno. Inspirada nas grandes lojas virtuais de games, o projeto transforma uma simples coleção de arquivos em uma experiência visual imersiva, moderna e altamente organizada.

## 🎯 Objetivo do Projeto

O objetivo principal foi criar um hub centralizado para o meu acervo pessoal. Em vez de navegar por pastas de arquivos confusas, a W3Labs Store oferece uma interface limpa, dividida por categorias, franquias e destaques, permitindo o download direto via integração com clientes torrent.

## ✨ Funcionalidades

- **Dashboard Intuitivo:** Banners de destaque, separação por categorias (Mundo Aberto, Simulação, Franquias específicas) e status de rede em tempo real.
- **Páginas de Detalhes Detalhadas:** Cada título possui um modal/página própria contendo:
  - Capa e imagens de demonstração.
  - Informações técnicas (Tamanho, Versão/Repack, Downloads, Classificação).
  - Sinopse completa do jogo/software.
  - Requisitos mínimos de sistema.
- **Integração com Torrent:** Botão "Abrir no Cliente Torrent" que aciona nativamente o software de download no computador via *magnet link*.
- **Ações Rápidas:** Opções de favoritar títulos e copiar o link de compartilhamento interno.
